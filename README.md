# Tesis Magíster en Hematología

## Autor
Raúl (poner nombre completo)

## Objetivo
Desarrollar análisis y modelos relacionados con hematología clínica y de laboratorio para la tesis de Magíster.

## Estructura del repositorio
- `data/` — datos crudos y procesados (no subir datos sensibles)
- `notebooks/` — Jupyter notebooks para exploración y visualización
- `src/` — código fuente y scripts reutilizables
- `results/` — figuras, tablas y salida de análisis
- `.venv/` — entorno virtual (ignorarlo en Git)

## Requisitos
Ver `requirements.txt` para dependencias del entorno.

## Uso rápido
1. Activar entorno:
```bash
source .venv/bin/activate
```
2. Ejecutar Jupyter:
```bash
jupyter notebook
```
